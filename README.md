# sztu-tools-kit

## 介绍

自己做的一些`SZTU(Shenzhen Technology University)`常用小工具的**源代码**.

本项目所有内容均为开源代码, 同时会附带打包好的运行程序, 如果你是普通学生, 可以直接下载对应工具使用; 如果你对源代码感兴趣, 可以clone本项目, 进行修改或者更多定制化内容!

感谢使用, 希望你 **Good morning Good evening and Good night!**
